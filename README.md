# 洛克王国孵蛋概率计算器

一个基于 KNN 的洛克王国孵蛋种类概率计算器。用户输入蛋的尺寸和重量后，程序会根据参考表中的尺寸/重量范围计算最近邻，并输出候选精灵的概率排序。

## 功能

- 输入蛋尺寸、蛋重量并调整 K 值
- 使用归一化距离和范围命中加权进行 KNN 预测
- 展示最可能孵化种类、概率、距离和命中的尺寸/重量范围
- 内置参考数据表，可按序列或名称搜索
- 纯静态前端，作者正在考虑部署到 EdgeOne Pages

## 本地运行

```bash
npm install
npm run dev
```

打开开发服务器显示的本地地址即可使用。

## 构建

```bash
npm run build
```

构建产物在 `dist` 目录。

## EdgeOne Pages 部署

- 框架：Vite
- 构建命令：`npm run build`
- 输出目录：`dist`
- Node 版本：建议 `20.15+`

## 数据说明

参考数据来自当前项目录入的孵蛋尺寸、重量范围表，该表源自网络。由于原始数据来自截图，后续如果有 Excel/CSV 原表，建议替换 `src/eggData.js` 以进一步校准名称和数值。
